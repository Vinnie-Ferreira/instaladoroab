# Calculadora CEDAE

Trata-se de uma calculadora que verifica a correta apuração da conta de água no município do Rio de Janeiro pela concecionária CEDAE.
Para tal, basta que o consumidor insira informações constantes na sua própria conta de água sem a necessidade de fornecer qualquer dado que o identifique.
As informações solicitadas são a forma de apuração, a primeira e a última medições, o número de dias medidos e o valor pago.
